## 版本说明

### 版本号规则
我们遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号**：当你做了不兼容的 API 修改
- **次版本号**：当你做了向下兼容的功能性新增
- **修订号**：当你做了向下兼容的问题修正

### 变更类型说明

- **新增** - 新功能
- **变更** - 对现有功能的变更
- **弃用** - 即将移除的功能
- **移除** - 已移除的功能
- **修复** - 问题修复
- **安全** - 安全相关的修复

### 发布周期

- **主版本**：根据重大功能和架构变更决定
- **次版本**：每月发布，包含新功能和改进
- **修订版本**：根据需要发布，主要是bug修复

### 支持政策

- **当前版本**：完全支持，包括新功能和bug修复
- **前一个主版本**：仅提供安全更新和关键bug修复
- **更早版本**：不再提供支持
