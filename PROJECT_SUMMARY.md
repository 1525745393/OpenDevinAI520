## 🚀 启动方式

### 一键启动
```bash
git clone https://github.com/1525745393/OpenDevinAI520.git
cd OpenDevinAI520
chmod +x start.sh && ./start.sh
```

### 多种模式
```bash
./start.sh          # 交互模式
./start.sh web       # Web界面模式
./start.sh cli       # 命令行模式
./start.sh both      # 双模式
./start.sh --help    # 查看帮助
```

### 状态检查
```bash
python check_status.py  # 检查项目状态
```
